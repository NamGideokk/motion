# MOTION

## 프로젝트 소개

- OOP + Type Sctipt 프로젝트
- Drag & Drop을 이용한 item 순서 변경

## 프로젝트 계획

- 일정 (6.28 ~ )

  - 기본 디자인 설계 & 마크업 & 스타일링
  - class를 이용한 객체 지향 프로그래밍 (OOP)
  - HTML Drag and Drop API 사용

- 기능

  - 필수 기능

    - 게시물 업로드 (Image, Video, Note, Todo-list)
    - 게시물 삭제

  - 넣고 싶은 기능

    - HTML Drag and Drop을 이용한 item 순서 변경

  - 부가 선택 기능

    - 회원제 (회원가입, 로그인, 회원탈퇴)
    - 소셜 계정 로그인 (구글, 카카오 등)
    - 게시물 수정
