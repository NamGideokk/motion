# MOTION



## 개요

- notion-simple-clone-web
- Drag & Drop을 이용한 item 순서 변경
- OOP + Type Sctipt 프로젝트



## 프로젝트 계획

- 일정 (6.28 ~ )
  - 기본 디자인 설계 & 마크업 & 스타일링
- 기능

  - 필수 기능
    - 게시물 업로드 (이미지, 비디오, 노트, Todo-list)
    - 게시물 삭제
  - 넣고 싶은 기능

    - Drag & Drop을 이용한 item 순서 변경

  - 부가 선택 기능
    - 회원제 (회원가입, 로그인, 회원탈퇴)
    - 소셜 계정 로그인 (구글, 카카오 등)
    - 게시물 수정
